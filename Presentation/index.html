A few terms to know

  • "Compile" = in terms of this tutorial compile is the generic word for minifying css/js, compiling js templates, and generating css from Less.
  • Task = a task is what is what you run. A task has one or more configurations
  • Package.json = the list of dependencies you need for your project. An example dependency might be grunt-contrib-watch
